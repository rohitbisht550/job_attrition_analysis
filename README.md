# job_attrition_analysis
Data analytics project showcasing employees attrition analysis using excel, python, sql and power Bi

📊 Job Attrition Analysis

📌 Project Overview

This project analyzes employee attrition data to identify key factors influencing employee turnover. The analysis is performed using SQL, Python, Excel, and Power BI to generate actionable business insights.

The goal is to help organizations reduce attrition and improve employee retention through data-driven decision-making.

🛠 Tools & Technologies Used

SQL – Data cleaning, querying, aggregations

Python (Pandas, Matplotlib, Seaborn) – Data analysis & visualization

Excel – Initial data exploration

Power BI – Interactive dashboard creation

📂 Project Files

Employee_Attrition_DataSet.csv → Raw dataset

job_attrition.sql → SQL queries for analysis

EA_project.ipynb → Python analysis notebook

JobAttritionDashboard.pbix → Power BI dashboard

Job Attrition Analysis project report.pdf → Detailed project report

Job-Attrition-Analysis.pptx → Presentation file

🔍 Key Analysis Performed

Attrition rate calculation

Department-wise attrition analysis

Salary & job role impact on attrition

Overtime vs attrition comparison

Years at company vs attrition

Gender-based attrition insights

📊 Key Insights

Employees working overtime show higher attrition.

Lower salary bands have higher turnover rates.

Engineering and Sales department shows comparatively higher attrition.

Employees with fewer years at the company are more likely to leave.

💡 Business Recommendations

Improve work-life balance policies.

Review compensation structure for lower salary bands.

Implement employee engagement programs.

Focus on retention strategies for high-risk departments.
